## 🏃‍♂️ Quick Start (3 Steps)

### Step 1: Install Dependencies

```bash
cd apps/mobile
npm install
```

### Step 2: Run Setup Script

```bash
./setup.sh
```

### Step 3: Start Development

```bash
# Start Metro bundler
npm start

# In another terminal - Run on device
npm run android  # For Android
npm run ios      # For iOS
```
